# BeatBerry 앱 아이콘 생성 프롬프트

## 목적
BeatBerry의 브랜드 방향(간결함, 빠른 변환, 친근한 베리 모티프)에 맞는 macOS 앱 아이콘 생성을 위한 프롬프트.

## 메인 프롬프트 (한국어)
아래 조건으로 macOS 앱 아이콘 스타일의 고해상도 이미지를 생성해줘.

- 앱 이름: BeatBerry
- 콘셉트: 오디오 포맷 변환 앱, 빠르고 간단한 변환 경험
- 핵심 모티프: 딸기(berry) + 오디오(파형, 음표, 스펙트럼) + 변환(순환 화살표/전환)
- 무드: 깔끔함, 신뢰감, 친근함, 현대적
- 스타일: 미니멀 3D 아이콘, 과도한 디테일 금지, 작은 크기에서도 식별 가능
- 색상: 딸기 레드(#E53935 계열), 잎사귀 그린(#43A047 계열), 포인트로 화이트/다크 그레이
- 배경: 투명 또는 단색 그라데이션(밝은 톤 우선)
- 구성: 중앙 정렬, 아이콘 내부 대비 강하게, 테두리 과장 금지
- 품질: 앱 스토어 아이콘 수준의 선명도, 노이즈/텍스트/워터마크 없음
- 출력 비율: 1:1
- 해상도: 최소 1024x1024

## Main Prompt (English)
Design a high-resolution macOS app icon for an app named "BeatBerry".

Requirements:
- Purpose: audio format converter app (fast, simple, reliable)
- Visual motif: strawberry + audio symbol (waveform/note/spectrum) + conversion cue (subtle circular arrows or transition mark)
- Style: clean minimal 3D icon, premium but friendly, no clutter
- Palette: strawberry red, leaf green, with white/dark gray accents
- Composition: centered subject, strong contrast, instantly recognizable at small sizes
- Background: transparent or simple bright gradient
- Quality: production-ready app icon quality, no text, no watermark, no noisy texture
- Aspect ratio: 1:1
- Resolution: at least 1024x1024

## 네거티브 프롬프트
- 복잡한 배경, 과도한 장식, 과한 반사 효과
- 글자 삽입(BeatBerry 텍스트 포함)
- 사진처럼 사실적인 과일 질감
- 저해상도, 흐림, 픽셀 깨짐, 워터마크

## 변형 프롬프트 3종
1. Flat Variant
- 플랫 벡터 느낌, 단순한 형태, 높은 식별성, 그림자 최소

2. Glass Variant
- 은은한 글래스 질감, 하이라이트 적당히, macOS 친화적인 세련된 느낌

3. Playful Variant
- 약간 더 귀여운 딸기 표정/실루엣 허용, 그러나 프로덕션 앱 수준의 정돈된 마감 유지
